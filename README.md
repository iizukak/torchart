# torchart

Experimental Art Projects with PyTorch

## Installation

```
$ pip3 install -r ./requirements.txt
$ export PYTHONPATH="$PWD:$PYTHONPATH"
```

## Usage

### Filter Visualization

```
$ python3 torchart/filtervisualization/main.py
```

### Run unit test

```
$ pytest ./torchart
```

### Check code format

```
$ flake8 ./torchart
```
