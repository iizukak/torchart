# torchart

Experimental Art Projects with PyTorch

## Install requirements

```
pip3 install -r ./requirements.txt
```

## Usage

### Filter Visualization

```
$ PYTHONPATH=. python3 torchart/filtervisualization/main.py
```

### Run unit test

```
$ pytest ./torchart
```

### Check code format

```
$ flake8 ./torchart
```
